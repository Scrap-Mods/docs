# Remove Joint

This packet is sent when the player removes a joint.

- **ID**: 0x27 (39)
- **Size**: 4 bytes
- **LZ4 Compressed**: Yes
- **State**: Playing
- **Bound To**: Client -> Server

## Structure

| Offset | Field Name | Field Type | Notes |
|--------|------------|------------|-------|
| 0x00   | Joint ID   | be u32     | The ID of the joint to remove. |
