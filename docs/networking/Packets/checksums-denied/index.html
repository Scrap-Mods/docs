Checksums Denied

If a checksum sent by File Checksums is invalid, this packet is sent.

  • ID: 0x08
  • Size: 5 bytes
  • LZ4 Compressed: Yes
  • State: Joining
  • Bound To: Client -> Server

Structure​

OffsetField NameField TypeNotes
0Indexuint32The index of the invalid checksum.
