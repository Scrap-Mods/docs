Generic Data (C->S)

To be documented.

  • ID: 0x1C (28)
  • Size: Variable
  • LZ4 Compressed: Yes
  • State: Playing
  • Bound To: Client -> Server

note

BlobData Payloads​

In this subsection, known BlobData payloads are listed. This is not guaranteed to be an exhaustive list.

Chat Message​

This BlobData is sent when the player sends a chat message. The server does not verify any of the fields and broadcasts them to all players without modification.

  • UID: 46968863-090a-46b8-ad99-1159b53450fe
  • Key: Always 0, stored as le u32.
  • World ID: No World
  • Flags: 0

Payload​

See 27 - Generic Data (S->C) - Chat Message.
