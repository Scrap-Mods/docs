# Character Info

Contains the name of the player and the customization options of the character.

- **ID**: 0x09
- **Size**: Variable
- **LZ4 Compressed**: Yes
- **State**: Joining
- **Bound To**: Client -> Server

## Structure

| Field Name              | Field Type | Notes |
|-------------------------|------------|-------|
| Name                    | [String](/docs/structures/networking/string) | The name of the player of the character. The server displays this name above the character and in the chat. |
| Character Customization | [Character Customization](/docs/structures/character-customization) | The customization options of the character. |
