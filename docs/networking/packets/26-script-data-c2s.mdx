# Script Data (C->S)

This packet is sent by the client to the server when the client sends a Lua network request.

- **ID**: 0x1A (26)
- **Size**: Variable
- **LZ4 Compressed**: Yes
- **State**: Playing
- **Bound To**: Client -> Server

## Structure

<BlobDataPacketsNotice />

| Field Name | Field Type | Notes |
|------------|------------|-------|
| Data       | [BlobData](../common-structures/blobdata.mdx)[until EOS] | The script data. This array is read until the end of the packet. |
