# Valid Checksums

If the checksums sent by [File Checksums](06-file-checksums.mdx) are valid, this packet is sent.

- **ID**: 0x07
- **Size**: 1 byte
- **LZ4 Compressed**: No
- **State**: Joining
- **Bound To**: Server -> Client

## Structure

No data.
