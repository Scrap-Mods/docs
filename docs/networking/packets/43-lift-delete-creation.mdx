# Lift Delete Creation

This packet is sent when the player deletes a creation from the lift.

- **ID**: 0x2B (43)
- **Size**: 4 bytes
- **LZ4 Compressed**: Yes
- **State**: Playing
- **Bound To**: Client -> Server

## Structure

| Offset | Field Name | Field Type | Notes |
|--------|------------|------------|-------|
| 0x00   | Lift ID    | be u32     | The ID of the lift delete the creation from. |

## Validation

If the lift does not exist, nothing happens. The server does not validate if the sender of this packet is the owner of the lift. Any player can delete creations of any lift.
