## sm.log

Used for logging information from scripts to the game log.

## Functions

### error

```lua
sm.log.error( ... )
```

Logs an error message.

<strong>Arguments:</strong> <br></br>

- <code>...</code> [<strong> any </strong>]: The arguments to display in the message.

---
