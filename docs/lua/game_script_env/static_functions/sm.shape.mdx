## sm.shape

**Associated object type:** [Shape](/docs/lua/game_script_env/userdata/Shape)

A <strong>shape</strong> is any block, part or basic material that can be built by a player. <br></br>
Shapes are always connected to a [body](/docs/lua/game_script_env/userdata/Body), which is a collection of shapes.

For more information about creating your own scripted shapes, see [ShapeClass](/docs/lua/game_script_env/classes/shape_class).

## Functions

### createBlock

```lua
sm.shape.createBlock( uuid, size, position, rotation, dynamic, forceSpawn )
```
<code>Server-Only</code> <br></br>

Creates a new block.

<strong>Arguments:</strong> <br></br>

- <code>uuid</code> [<strong> <a href="/docs/lua/game_script_env/userdata/Uuid"> Uuid </a> </strong>]: The block uuid.
- <code>size</code> [<strong> <a href="/docs/lua/game_script_env/userdata/Vec3"> Vec3 </a> </strong>]: The block size.
- <code>position</code> [<strong> <a href="/docs/lua/game_script_env/userdata/Vec3"> Vec3 </a> </strong>]: The the world position.
- <code>rotation</code> [<strong> <a href="/docs/lua/game_script_env/userdata/Quat"> Quat </a> </strong>]: The world rotation (defaults to none).
- <code>dynamic</code> [<strong> bool </strong>]: Whether the shape is dynamic or static. Defaults to true (dynamic).
- <code>forceSpawn</code> [<strong> bool </strong>]: Whether to force spawn the shape or not. Optional.

<strong>Returns:</strong> <br></br>

- [<strong> <a href="/docs/lua/game_script_env/userdata/Shape"> Shape </a> </strong>]: The created shape.
