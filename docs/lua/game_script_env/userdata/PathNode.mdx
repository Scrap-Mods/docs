## PathNode

**Associated namespace:** [sm.pathNode](/docs/lua/game_script_env/static_functions/sm.pathNode)

A userdata object representing a <strong>path node</strong> in the game.

<strong>Operations:</strong>

| Operation   | Description |
| ----------- | ----------- |
| <code>PathNode</code> == <code>PathNode</code> | Checks if two instances of <code>PathNode</code> refer to the same <code>PathNode</code>. |


## Functions

### connect

```lua
pathNode:connect( target )
```
<code>Server-Only</code> <br></br>

Creates a PathNode connection.

<strong>Arguments:</strong> <br></br>

- <code>pathNode</code> [<strong> <a href="/docs/lua/game_script_env/userdata/PathNode"> PathNode </a> </strong>]: The pathNode.
- <code>target</code> [<strong> <a href="/docs/lua/game_script_env/userdata/PathNode"> PathNode </a> </strong>]: The pathNode to connect to.
