### GameClass
A script class that defines the game mode. Only one instance of this class is made.

This is the first script that will be run.

The game script is responsible for creating and managing [Worlds](/docs/lua/game_script_env/userdata/World).

The class can receive events sent with [sm.event.sendToGame](/docs/lua/game_script_env/static_functions/sm.event#sendtogame).

The fields below are accessed using <code>self.fieldName</code> in the GameClass script:

<strong>Fields:</strong>

| Type        | Name           | Description |
| ----------- | -----------    | ----------- |
| [Network](/docs/lua/game_script_env/userdata/Network) | network | A [Network](/docs/lua/game_script_env/userdata/Network) object that can be used to send data between client and server. |
| [Storage](/docs/lua/game_script_env/userdata/Storage) | storage | A server-side [Storage](/docs/lua/game_script_env/userdata/Storage) object that can be used to save and load data to/from the world database. |
| any | data | Game start data. |

## Server-only

### onPlayerJoined

```lua
function GameClass.server_onPlayerJoined( self, player, newPlayer )
end
```
Called when a [Player](/docs/lua/game_script_env/userdata/Player) joins the game. <br></br>

<strong>Arguments:</strong> <br></br>

- <code>self</code> [<strong> table </strong>]: The class instance.
- <code>player</code> [<strong> <a href="/docs/lua/game_script_env/userdata/Player"> Player </a> </strong>]: The joining player.
- <code>newPlayer</code> [<strong> bool </strong>]: True if the player has not been in this game before.

### onLanguageChange

```lua
function GameClass.client_onLanguageChange( self, language )
end
```
Called when the user changes the language in the in-game menu.

Valid languages are:
<code>Brazilian</code>, <code>Chinese</code>, <code>English</code>, <code>French</code> <br></br>
<code>German</code>, <code>Italian</code>, <code>Japanese</code>, <code>Korean</code> <br></br>
<code>Polish</code>, <code>Russian</code>, <code>Spanish</code> <br></br>

<strong>Arguments:</strong> <br></br>

- <code>self</code> [<strong> table </strong>]: The class instance.
- <code>language</code> [<strong> string </strong>]: The new language.
